# Immigration Country Matcher

Интерактивный прототип навигатора стран для переезда граждан РФ.

## Где находятся данные

Данные вынесены из `index.html` в отдельный файл:

- `data/countries.js` — каталог 249 стран и территорий ISO;
- 11 стран имеют заполненную исследовательскую модель и участвуют в рейтинге;
- остальные страны имеют статус `pending` и не получают вымышленные баллы.

Для каждой исследованной страны хранятся:

- нормализованные значения для рейтинга;
- сроки до ПМЖ и гражданства;
- бюджет, школа, семья, медицина, налоги, инфраструктура;
- отдельные поля для собак, кошек и других животных;
- подробные разделы карточки;
- ссылки на официальные источники;
- дата обновления.

## Интерфейс

- расчёт выполняется только по кнопке;
- карточка страны открывается по клику;
- в карточке показывается персональная оценка, подробная информация и источники;
- полный каталог доступен через поиск.

## GitHub Pages

Сайт публикуется из ветки `main`, папка `/`.

## Статус

Версия 0.4 — архитектурный MVP. Для коммерческого запуска требуется заполнить полные исследования всех стран и регулярно перепроверять данные.


## Изменения версии 0.4

- удалён вопрос о максимальной стоимости школы;
- все поля выбора имеют одинаковую высоту;
- готовность к языковому экзамену влияет только при цели ПМЖ или гражданства;
- при цели «ВНЖ достаточно» языковой ответ не снижает рейтинг;
- стоимость школы остаётся в карточке страны как справочная информация.

- модель данных поддерживает отдельные поля `language_for_pr` и `language_for_citizenship`; до их заполнения используется прежний общий флаг как временная подстраховка.
