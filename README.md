# Immigration Country Matcher

Интерактивный прототип навигатора стран для переезда граждан РФ.

## Текущая версия

Версия 0.2:

- единая прокрутка страницы;
- расчёт только по кнопке;
- разделены подтверждаемый доход и бюджет жизни;
- гражданство задаётся как цель, а не как желаемое количество лет;
- добавлен выбор вида домашнего животного;
- уточнена формулировка использования английского;
- создана расширенная схема исследования для разных категорий граждан РФ.

## Важное ограничение

Текущие данные стран первоначально собирались для одной конкретной семьи. Они ещё не являются универсальной коммерческой базой для всех миграционных профилей.

Полный перечень данных, которые необходимо исследовать, находится в `research-schema.md`.
