# Immigration Country Matcher

Интерактивный прототип персонального матчера стран для переезда граждан РФ.

## Рабочий пилот Испании

Новая вертикальная версия доступна по адресу:

- `https://sankhipkate.github.io/immigration-country-matcher/pilot/`

Она выполняет полный сценарий:

1. собирает один профиль пользователя;
2. проверяет все шесть миграционных маршрутов Испании;
3. использует отдельный порядок строгости внутри маршрута и порядок предпочтительности между маршрутами;
4. выбирает лучший маршрут;
5. отдельно оценивает доход, семью, ПМЖ и гражданство, работу, бюджет, города и открытые пробелы данных;
6. показывает официальные источники, на которых основан результат.

## Данные

- `data/catalog.js` — каталог 249 стран и территорий ISO;
- `data/researched.js` — прежняя сравнительная модель 11 исследованных стран;
- `data/spain-research-v2.2.json` — структурированное исследование Испании;
- `data/research-package-v2.2.schema.json` — строгая JSON Schema исследовательского пакета.

Испанский пакет содержит шесть маршрутов, три города, атомарные факты, источники и открытые вопросы. Статус исследования — `SECONDARY_REVIEW`.

## Расчётный движок

`js/spain-calculator.js` экспортирует:

- `calculateSpain(profile, data)`;
- `resolveStatusConflict(statuses)`;
- `selectBestVariant(variants)`.

Используются шесть канонических статусов маршрута:

- `SUITABLE`;
- `SUITABLE_WITH_CONDITIONS`;
- `PRELIMINARY_SUITABLE`;
- `INSUFFICIENT_COUNTRY_DATA`;
- `INDIVIDUAL_REVIEW_REQUIRED`;
- `UNSUITABLE`.

Практическое соответствие города не смешивается со статусом миграционного маршрута.

## Тестирование

```bash
npm test
```

Тесты используют встроенный `node:test` и проверяют:

- конфликт статусов внутри маршрута;
- выбор лучшего варианта;
- конвертацию USD в EUR;
- DNV, NLV и highly qualified;
- российские банковские документы;
- конфликт цели гражданства с обязательным сохранением гражданства РФ;
- отдельный практический результат бюджета города.

GitHub Actions запускает тесты при каждом pull request и push в `main`.
