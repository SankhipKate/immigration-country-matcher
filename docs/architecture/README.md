# Архитектура универсального Immigration Matcher

Файлы этого этапа:

1. `UNIVERSAL_PROFILE_V1.md` — человекочитаемый контракт единого профиля.
2. `QUESTIONNAIRE_PROFILE_MAP_V1.md` — карта всех 29 вопросов утверждённой анкеты.
3. `COUNTRY_ENGINE_REFACTOR_V1.md` — техническая спецификация перехода от `calculateSpain()` к универсальному движку.
4. `TWO_COUNTRY_MVP_PLAN.md` — порядок создания MVP Испания + Уругвай.
5. `../../data/schemas/user-profile-v1.schema.json` — строгая JSON Schema Draft 2020-12.
6. `../../tests/fixtures/universal-profile-samples-v1.json` — валидные тестовые профили.

Зафиксированные решения:

- Уругвай — вторая страна MVP;
- основная анкета не содержит вопросов конкретной программы;
- валютный курс не вводится пользователем;
- все варианты легализации оцениваются независимо;
- выбранный тип дохода не назначает маршрут заранее;
- юридическая и практическая оценки разделены;
- отдельный калькулятор Уругвая не создаётся.
