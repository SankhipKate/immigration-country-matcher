# Карта анкеты MVP → универсальный профиль v1

Эта карта фиксирует реализацию всех 29 вопросов основной анкеты. Она не добавляет новую испанскую или уругвайскую анкету.

| ID | Вопрос | Путь в профиле | Обязательность | Влияние |
|---|---|---|---|---|
| Q01 | У вас есть гражданство РФ? | `citizenships: ["RU"]` | обязательный входной фильтр | при ответе «Нет» анкета не открывается; другие гражданства не учитываются |
| Q02 | В какой стране вы сейчас живёте? | `residence.current_country` | обязательно | место подачи, консульская юрисдикция |
| Q03 | Какой у вас статус в этой стране? | `residence.current_status` | обязательно | право подаваться из текущей страны |
| Q04 | Какие способы подачи готовы рассматривать? | `application_preferences.methods[]` | обязательно | исключение недоступных способов подачи |
| Q05 | Кто переезжает вместе с вами? | `family.adults_count`, `family.partner_included`, `family.children[]` | обязательно | семейная доступность и финансовая формула |
| Q06 | Как оформлены отношения? | `family.relationship_type` | при наличии партнёра | признание партнёра и семейный ВНЖ |
| Q07 | Сколько детей переезжает? | длина `family.children[]` | при наличии детей | финансовые надбавки, семейный ВНЖ |
| Q08 | Возраст каждого ребёнка | `family.children[].age_years` | при наличии детей | предел возраста иждивенца |
| Q09 | Нужна ли детям школа? | `family.school_needed` | при наличии детей | только практическая оценка городов и школ |
| Q10 | Учитывать ЛГБТ-права и признание семьи? | `lgbt.enabled`, `lgbt.consent_for_personalization` | обязательно | отдельный семейный и практический трек |
| Q11 | Основной источник дохода | `income.primary.type` | обязательно | совместимость дохода со всеми вариантами, не выбор маршрута |
| Q12 | Страна источника дохода | `income.primary.source_country` | обязательно | иностранный/местный/российский доход |
| Q13 | Страна банковского счёта | `income.primary.bank_country` | обязательно | допустимость банковских документов |
| Q14 | Есть ли другие источники дохода? | `income.has_additional_sources` | обязательно | необходимость уточняющего блока |
| Q15 | Подтверждаемый доход заявителя | `income.primary.monthly_provable.amount`, `.currency` | обязательно | проверка финансовых требований |
| Q16 | Полнота подтверждения дохода | `income.primary.evidence_level` | обязательно | надёжность доказательств дохода |
| Q17 | Есть ли доход у партнёра? | `income.partner.has_income` | при наличии партнёра | возможность суммирования доходов |
| Q18 | Доход партнёра | `income.partner.sources[].monthly_provable` | при наличии дохода | семейная финансовая формула |
| Q19 | Подтверждение дохода партнёра | `income.partner.sources[].evidence_level` | при наличии дохода | допустимость дохода партнёра |
| Q20 | Какой долгосрочный результат необходим? | `goal.long_term` | обязательно | ВНЖ, ПМЖ, гражданство |
| Q21 | Сколько времени готовы жить в стране? | `goal.physical_presence` | обязательно | сохранение статуса, ПМЖ, гражданство |
| Q22 | Готовность к языковому экзамену | `goal.language_exam_readiness` | обязательно | только ПМЖ/гражданство, если язык требуется |
| Q23 | Важность сохранения гражданства РФ | `goal.keep_russian_citizenship` | обязательно | долгосрочная совместимость |
| Q24 | Максимальный бюджет семьи | `preferences.monthly_budget` | обязательно; может быть `null` | только практическая пригодность |
| Q25 | Размер города | `preferences.city_size` | обязательно | выбор городов |
| Q26 | Климат | `preferences.climate` | обязательно | выбор городов |
| Q27 | Какие животные переезжают? | `pets.types[]` | обязательно | практические правила ввоза и содержания |
| Q28 | Порода собаки | `pets.dogs[].breed` | при наличии собаки; допускается `null` | только подтверждённые породные ограничения |
| Q29 | Особые миграционные обстоятельства | `special_circumstances[]` | обязательно | индивидуальная проверка, не автоотказ |

## Уточняющие блоки после первого результата

| Блок | Данные | Путь |
|---|---|---|
| Дополнительные источники | тип, страна, банк, сумма, валюта, подтверждение | `income.additional_sources[]` |
| История дохода | длительность, стабильность, сохранение после переезда | поля `incomeSource.history_months`, `stability`, `continues_after_move` |
| Договор или бизнес | остаток срока договора, возраст бизнеса | `contract_remaining_months`, `business_age_months` |
| Накопления | сумма и валюта | `income.savings` |
| Медицина | конкретное лекарство или регулярный врач | `optional_modules.medical` |
| Вариант легализации | специфические ответы программы | `route_specific_answers[route_id]` |

## Запрещённые подмены

| Нельзя | Почему | Правильно |
|---|---|---|
| `legalResidence: true/false` как вопрос пользователю | пользователь не обязан знать юридическое значение резидентства | спросить фактический статус и вычислить применимость |
| `plannedBasis → ES_DNV` | заранее назначает победивший вариант | проверять все варианты по фактам профиля |
| ручной `eurUsdRate` | пользователь не обязан знать курс | курс в расчётном контексте с источником и датой |
| «готовность к испанскому» | неясно: знание или желание учить | готовность к экзамену для ПМЖ/гражданства |
| бюджет без определения | непонятно, что входит | максимальные расходы семьи с явным составом расходов |
| обязательный вопрос о лекарстве | практическая деталь превращается в ядро анкеты | необязательный медицинский модуль |
| порода собаки как юридический фильтр | порода обычно не относится к ВНЖ | проверять только официальные ограничения на ввоз/содержание |
